Guide de l'utilisateur de l'application PowerTrain

Paramètres

La page des paramètres vous permet de configurer divers aspects de l'application pour personnaliser votre expérience. Voici les principaux champs de données qu'elle peut contenir :

  • FTP Cyclisme : exprimé en Watt.
  • FTP Aviron : exprimé en min/500m.
  • Mode développeur : permet l'activation de certaines fonctionnalités cachées pour le débogage.

Les valeurs FTP sont utilisées pour calculer les cibles de puissance pour les sessions d'entraînement.

Recherche d'appareils

La page de scan liste les appareils Bluetooth disponibles, comme les machines de fitness et les capteurs. Les appareils suivants sont pris en charge : machines compatibles FTMS (comme les rameurs et les home trainers), moniteurs de fréquence cardiaque et capteurs de cadence (uniquement pour les home trainers)

Recherche d'appareils

  • Appuyez sur le bouton "Scanner les appareils" pour rechercher les machines de fitness et capteurs Bluetooth à proximité.
  • L'application affichera une liste des appareils découverts, incluant les rameurs, les vélos d'appartement et les moniteurs de fréquence cardiaque.
  • Appuyez sur le bouton "Connecter" près de l'appareil pour l'associer à votre téléphone.

Se connecter à Strava

  • Connectez votre compte Strava pour synchroniser vos entraînements.
  • Appuyez sur le bouton de connexion Strava pour authentifier et lier votre compte.
  • Une fois connecté, vos sessions d'entraînement, une fois terminées, peuvent être téléchargées sur Strava.

Démarrage et réalisation d'une session d'entraînement

Démarrage d'une session

  • Assurez-vous que votre machine de fitness et vos capteurs sont connectés.
  • Appuyez sur le bouton "Ouvrir" près du nom de la machine connectée.
  • En haut, vous verrez une liste d'itinéraires prédéfinis. Vous pouvez en sélectionner un optionnellement. Si vous ne le faites pas, l' application en choisira un au hasard.
  • Vous pouvez choisir de démarrer un "Trajet libre", générer une session d'entraînement ou sélectionner une "Session d'entraînement" sauvegardée
  • Trajet libre :
    • Sélectionnez "Temps" ou "Distance".
    • Sélectionnez la durée ou la distance souhaitée.
    • Optionnellement, pour les rameurs, vous pouvez sélectionner le rythme de nage et les cibles de rythme.
    • Optionnellement, pour les trainers vous pouvez sélectionner la puissance cible et la cadence.
    • Si la machine prend en charge le réglage de la résistance, vous pouvez sélectionner le niveau de résistance du rameur/trainer dans sa plage acceptée.
    • Appuyez sur "Démarrer".
  • Générateur de session d'entraînement :
    • Sélectionnez la durée souhaitée.
    • Sélectionnez le type d'entraînement.
    • Si la machine prend en charge le réglage de la résistance, vous pouvez sélectionner le niveau de résistance du rameur/trainer dans sa plage acceptée.
    • Appuyez sur "Démarrer".
  • Session d'entraînement sauvegardée :
    • Sélectionnez une session d'entraînement dans la liste.
    • Appuyez sur "Démarrer la session".

Réalisation d'une session

En haut, vous avez le titre de la session et quelques contrôles permettant de couper le son du métronome, le son des alertes, de mettre en pause/démarrer et arrêter la session.

Une fois démarrée, vous verrez en haut une représentation visuelle des intervalles et le temps écoulé actuel et restant ou la distance.

À gauche, vous trouverez l'intervalle actuel avec ses cibles et le temps écoulé et restant ou la distance.

La partie principale est remplie avec les données réelles provenant de la machine de fitness et des capteurs. Les valeurs cibles, telles que définies dans la session d'entraînement, changent de couleur en rouge lorsqu'elles ne sont pas dans la plage souhaitée.

Il y a aussi une carte superposée avec votre position actuelle et l'itinéraire.

Lorsque l'intervalle actuel est sur le point de se terminer, un bip est joué pour vous alerter.

Démarrage d'une session

Commencez à ramer ou pédaler

Si l'intervalle actuel a une cible de rythme de nage, alors vous entendrez un son de métronome au rythme de nage cible. Vous pouvez désactiver ce son en appuyant sur l'icône du métronome.

Mise en pause/reprise d'une session

Arrêtez de ramer ou pédaler. S'il n'y a pas d'activité, l'application mettra automatiquement la session en pause.

Ou, vous pouvez appuyer sur le bouton pause/reprise.

Fin d'une session

Ramez/pédalez jusqu'à la fin de la session. Vous aurez alors trois options :

  • Sauvegarder la session. Si connecté à Strava, la session est téléchargée automatiquement sur Strava. Vous la trouverez là avec la carte d'itinéraire correspondante. Si pas connecté à Strava, vous trouverez l'activité sous le menu "Activités non synchronisées"
  • Continuer la session : la session sera prolongée pour que vous puissiez continuer à ramer/pédaler avec les mêmes paramètres que le dernier intervalle.
  • Abandonner la session : toutes les données sont perdues

Vous pouvez appuyer sur le bouton "Arrêter" à tout moment pour terminer la session.

Note sur les vélos d'intérieur (home trainers)

Votre home trainer doit être un trainer intelligent et prendre en charge le réglage de la puissance cible (mode ERG). Les sessions sont réalisées en mode ERG. L'application définit la puissance cible. Le trainer ajuste continuellement la résistance pour que vous restiez à cette puissance, même si votre cadence de pédalage change.

Visualisation et édition des sessions d'entraînement

Sélectionnez "Sessions d'entraînement" dans le menu burger, puis sélectionnez le type de machine dans la liste.

Il y a des sessions d'entraînement intégrées et des sessions d'entraînement personnalisées. Les sessions intégrées ne peuvent pas être éditées mais peuvent être dupliquées.

Vous pouvez ajouter, éditer et supprimer des sessions d'entraînement personnalisées. Elles sont stockées localement sur votre appareil mobile.

Lorsque vous créez d'abord une session d'entraînement personnalisée, vous pouvez sélectionner si la session est basée sur le temps ou la distance basée. Une fois sélectionnée et la session sauvegardée, vous ne pouvez pas changer son type.

Vous pouvez ajouter, éditer et supprimer des "intervalles unitaires" et des "intervalles de groupe".

Les intervalles unitaires sont des intervalles avec une durée ou une distance, selon le type de session. Les intervalles de groupe sont des intervalles contenant un ou plusieurs intervalles unitaires avec un nombre de répétitions.

Les intervalles unitaires ont un titre, une durée/distance et des cibles. Vous pouvez choisir d'avoir une cible pour la vitesse/puissance, ou la cadence.

La vitesse/puissance est exprimée en pourcentage et ses valeurs sont calculées en fonction de la valeur FTP que vous avez définie dans les paramètres.

Note sur le niveau de résistance : Pour les machines d'aviron, cela dépend de la machine. Les rameurs Domyos ont 15 niveaux de résistance et les valeurs sont 10, 20, 30, ..., 150 pour les résistances de 1 à 15. Vous pouvez jouer avec le mode développeur pour découvrir les niveaux de résistance de votre machine. Aussi, en mode trajet libre, vous pouvez sélectionner le niveau de résistance tel qu'il est pris en charge par votre machine.

Activités non synchronisées

Sélectionnez "Activités non synchronisées" dans le menu burger.

Vous trouverez là les fichiers d'entraînement générés par l'application lorsqu'ils n'ont pas pu être envoyés à Strava.

Pour chacun, vous pouvez essayer de le télécharger sur Strava, le supprimer, ou le partager avec d'autres applications.

Appuyez sur le nom de l'activité. Vous verrez une représentation visuelle des données contenues dans le fichier, comme la vitesse/rythme, la cadence et la fréquence cardiaque

Si vous êtes connecté à Strava et que les téléchargements n'ont pas échoué, la liste devrait être vide.

Mode développeur

Lorsque vous activez le mode développeur dans les paramètres, vous aurez accès aux fonctionnalités prises en charge par la machine de fitness et vous pouvez lire et envoyer des données de et vers la machine.

Avertissement : Utilisez ceci avec prudence !

Depuis la page de scan, sélectionnez la machine et appuyez sur "Ouvrir". Vous verrez deux menus en bas de la page : "Fonctionnalités de données d'appareil" et "Fonctionnalités de machine".

Fonctionnalités de données d'appareil

Voici les données qui peuvent être lues depuis la machine, avec les valeurs actuelles.

Fonctionnalités de machine

Voici les commandes qui peuvent être envoyées à la machine. Parmi elles, vous pourriez pouvoir définir le "niveau de résistance" ou la cible de puissance (mode ERG). Lorsque vous voulez utiliser l'une de celles-ci, envoyez toujours d'abord la commande de contrôle de demande.
